### Requirents Technic
- Applicação foi feita utilizando-se PHP: ^7.4
- Banco de dados sql server
- Necessários os drives para a conexeção com o sql server

## RUN THE PROJECT - API
- Configure .env com a configuração para se conectar no banco de dados
- Rode os comandos:
  - ```console
       composer install
       composer up-tables
       ```
